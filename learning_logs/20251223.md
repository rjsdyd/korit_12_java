## Scanner
1. Scanner scanner = new Scanner
- 객체 생성
2. Scanner.nextLine() 
- 메서드 호출
3. nextLine()
- 특정 라인이 실행됐을 때 데이터를 입력받지 않고 그대로 넘어가서 실행된다는 점은 nextLine()의 문제점이다. 이를 해결하는 방법으로는 Enter키를 받아주는 nextLine()을 하나 더 만드는 것이다.
ex. 
System.out.print("올해는 몇 년도 인가??");
int year = scanner.nextInt();
scanner.nextLine();
-> 해결방법

## Conversion
1. Upcasting (업캐스팅) : 더 큰 용량의 자료형으로 변환시키는 것.
- 방법 #1 : 형 변환을 하기위해서는 '(바꿀데이터타입) 변수명'으로 작성
ex. 
char cast1 = 'A';
System.out.println("변형 : " + (int)cast1);
- 방법 #2 : 새로운 변수를 선언하여 캐스팅이 이루어진 데이터를 집어넣는 방법
ex.
char cast3 = 'b';
int casted3 = cast3;
System.out.println("원형 : " + cast3);
System.out.println("변형 : " + casted3);
2. Downcasting (다운캐스팅) : 용량이 더 작은 자료형으로 변형하는 것.
